أريد إصلاح مشكلة عرض القيم الحالية داخل لوحة الأدمن، خصوصًا في:
- /admin/packages/:id/edit
- /admin/settings

المطلوب باختصار:
أريد كل الحقول تعرض القيم الحالية المحفوظة فعليًا، بحيث أقدر أعدل عليها مباشرة، بدل ما تبدو فارغة.

==================================================
1) إصلاح صفحة تعديل الباقة /admin/packages/:id/edit
==================================================
المشكلة الحالية:
- اسم الباقة يظهر في العنوان بشكل صحيح
- بعض بيانات تبويب "المحتوى" تظهر جزئيًا
- لكن أغلب حقول تبويب "أساسي" و"يشمل" و"خيارات" تبدو فارغة
- هذا يمنعني من تعديل الباقة بسهولة

المطلوب:
- عند فتح صفحة تعديل أي باقة، اجلب بيانات الباقة الحالية من الـ API
- اعمل prefill لكل الحقول داخل كل التبويبات، وليس جزءًا منها فقط

يجب تعبئة هذه الحقول بالقيم الحالية:
- slug
- emoji / icon
- titleAr / titleEn
- shortDescriptionAr / shortDescriptionEn
- fullDescriptionAr / fullDescriptionEn
- category
- minPrice / maxPrice
- durationAr / durationEn
- badgeAr / badgeEn
- primaryColor / badgeColor
- rating
- reviewCount
- sortOrder
- cancellationPolicyAr / cancellationPolicyEn
- status
- includes / excludes
- whyThisTrip
- itinerary
- faq
- whatToBring
- suitableFor
- images
- videos
- أي حقول أخرى موجودة في package model

راجع السبب الجذري وأصلحه:
- إذا كان fetch يعمل لكن form لا يتحدث بعد وصول البيانات، أصلحه
- إذا كنت تستخدم react-hook-form، استخدم reset(fetchedData) بعد الجلب
- إذا كان هناك mapping ناقص بين response وform state، أصلحه
- إذا كانت بعض التبويبات لها state منفصلة، تأكد أنها أيضًا تتعبى من نفس البيانات
- لا تجعل EMPTY_PKG أو initial state يمسح القيم بعد التحميل

مهم جدًا:
- لا أريد فقط عنوان الباقة في أعلى الصفحة
- أريد القيم داخل جميع الـ inputs وtextareas وselectors نفسها

==================================================
2) إصلاح صفحة الإعدادات /admin/settings
==================================================
المشكلة الحالية:
- القيم تبدو وكأنها موجودة لكن النص داخل الحقول باهت جدًا أو شبه مختفٍ
- الشكل الحالي يجعلها تبدو فارغة رغم أنها محملة

المطلوب:
- اجعل القيم الحالية ظاهرة بوضوح داخل الحقول
- لا تجعل الـ inputs تبدو disabled إذا كانت قابلة للتعديل
- أصلح:
  - text color
  - opacity
  - placeholder styling
  - disabled/readOnly styling لو مطبق بالخطأ
  - background contrast
- إذا كانت الحقول editable، يجب أن تبدو editable بصريًا
- إذا كانت readOnly مؤقتًا، أوضح ذلك بصريًا بدل أن تبدو فارغة

أريد أن تكون قيم الإعدادات الحالية مقروءة بوضوح مثل:
- اسم الشركة
- الموقع
- أرقام التواصل
- روابط السوشيال
- العملة الافتراضية
- أسعار الصرف
- hero title / subtitle
- feature toggles

==================================================
3) راجع الـ CSS / UI states
==================================================
أريدك أن تراجع هل يوجد:
- opacity منخفضة على input text
- text color قريب من لون الخلفية
- disabled style مطبق على حقول editable
- readOnly style غير مناسب
- autofill / controlled input issue
- focus state فقط هي التي تُظهر القيمة بدل الوضع العادي

==================================================
4) loading / error states
==================================================
أضف:
- loading واضح أثناء تحميل البيانات
- error message واضحة إذا فشل الجلب
- لا تجعل الصفحة تبدو فارغة بدون تفسير

==================================================
5) اختبار مطلوب
==================================================
اختبر فعليًا:
- افتح /admin/packages/1/edit وتأكد أن كل الحقول مليانة بالقيم الحالية
- افتح /admin/settings وتأكد أن القيم الحالية ظاهرة بوضوح وليست باهتة
- عدل قيمة واحدة في الباقة واحفظ ثم أعد فتح الصفحة وتأكد أن القيمة ظهرت
- عدل قيمة واحدة في الإعدادات واحفظ وتأكد أنها تظل ظاهرة بعد reload

==================================================
6) المطلوب بعد التنفيذ
==================================================
بعد الانتهاء أعطني ملخصًا واضحًا:
- لماذا كانت حقول تعديل الباقة فارغة
- لماذا كانت قيم الإعدادات تبدو غير ظاهرة
- ما الملفات التي تم تعديلها
- كيف تم إصلاح prefill ووضوح القيم
