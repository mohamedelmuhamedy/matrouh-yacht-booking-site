عاوزك تصلح النقاط دي فقط في لوحة الأدمن والموقع العام، وتختبرهم بنفسك end-to-end:

1) Hero default / restore defaults
- خلي الشكل الحالي الصحيح هو الـ default baseline
- المقصود: عنوان الهيرو يظهر:
  - "اكتشف جمال" باللون الأبيض
  - "مرسى مطروح" باللون الـ gradient الحالي الصحيح
- زر "استعادة الأصل" لازم يرجّع لهذا الشكل بالضبط، وليس للشكل البايظ القديم
- لو فيه default values قديمة غلط، حدّثها في DB / seed / fallback

2) Feature switches
- التلات switchات:
  - المساعد الذكي
  - مقارنة الباقات
  - قسم التقييمات
لازم يشتغلوا فعليًا
- عند إغلاق أي واحد:
  - يختفي من الموقع العام فعلًا
- عند فتحه:
  - يظهر تاني
- لازم القيم تتحفظ في DB كـ boolean حقيقي
- وبعد refresh يفضلوا بنفس الحالة

3) Packages pricing logic
- في صفحة تعديل الباقة:
  - لو "الحد الأقصى" فاضي => يبقى مفيش حد أقصى
  - وقتها الموقع يعرض السعر كقيمة واحدة فقط، وليس range
- لو "الحد الأقصى" مكتوب => اعرض الحد الأدنى والحد الأقصى معًا
- لا تجبر maxPrice على 0 أو string فارغ يبوظ العرض
- خلي maxPrice nullable في DB والـ API والـ form mapping

4) Cancellation policy toggle per package
- أضف switch في تعديل الباقة:
  - "تفعيل سياسة الإلغاء"
- لو غير مفعل:
  - لا تظهر خانة سياسة الإلغاء في الأدمن
  - ولا تظهر سياسة الإلغاء في صفحة الرحلة العامة
- لو مفعل:
  - تظهر ويمكن تعديلها
- لازم تتحفظ حالتها في DB

5) Logo management
- أضف في الإعدادات إمكانية:
  - رفع لوجو جديد
  - تغيير اللوجو الحالي
  - معاينة اللوجو
- اللوجو الجديد لازم يشتغل فعليًا في:
  - الـ navbar
  - الفوتر
  - أي مكان يستخدم اللوجو
- احفظه في storage/db بشكل صحيح
- لو لا يوجد لوجو مرفوع، استخدم fallback واضح

6) Testing
اختبر بنفسك بعد التنفيذ:
- restore defaults
- التلات switches وتأثيرهم على الموقع العام
- max price فارغ / غير فارغ
- cancellation policy on/off
- logo upload/change
- save + refresh

7) بعد الانتهاء:
أعطني ملخصًا قصيرًا فقط:
- ما الذي أصلحته
- ما الذي كان سبب المشكلة
- وهل اختبرت كل نقطة بنفسك أم لا
- وابعت Screenshots للنقاط التالية:
  - الهيرو بعد restore الصحيح
  - switches بعد حفظها
  - باقة بدون حد أقصى
  - باقة بحد أدنى وأقصى
  - تغيير اللوجو
